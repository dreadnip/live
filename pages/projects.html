List of pet projects

  • Mono

    A tiny, single-file PHP framework that I use for small projects. Implemented as a PSR-15 middleware stack application, it offers routing, DI and templating all in +- 200 LOC.

  • Jug

    Very simple 1-to-1 Twig to HTML static site generator in PHP.

  • Static

    Proof-of-concept for bundling a PHP micro SAPI with a phar compiled with Box into a single binary.

  • PHPStan demo

    A teaching repo that has basic errors for each level of PHPStan.

  • Simple payments

    Study project on how to implement Mollie payments in a vanilla PHP way.

  • VRTlite

    A text-only version of vrt.be/vrtnws implemented as a Symfony application with a Redis cache

  • Chrome PDF bundle

    Symfony bundle to generate PDFs with headless Chrome using chrome-php/chrome.

  • Slides

    Pure HTML/CSS/JS slides for presentations.

  • Chop

    Reset the OPcache content from the CLI, without reloading PHP-FPM

  • Pocket

    Study project on Reflection based PHP ORMs.

  • Smart DTO bundle

    Map your Doctrine entities into data transfer objects for faster usage in Symfony forms

  • Highlighter

    Customizable PHP syntax highlighter using tokens

  • Form storage

    Self-hosted Formspree alternative. Takes POST inputs and stores them in an SQLite db.

  • Mailgun logger

    A basic PHP class to quickly access (and store) Mailgun logs.

  • Kanban board

    A basic kanban board in a single HTML file using browser native drag & drop and localStorage for persistence.
