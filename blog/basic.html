Technology I'm building with right now

2023 was an interesting year for web dev. Starting 2024, this is the tech stack that I'm using to build my apps

Dropping JS & CSS build steps

Honestly, I never cared for the build step. It always seemed like a thick layer of complexity for some small upside

But vanilla CSS has been improving, browser support is great for all modern features, and we have importmaps now for JS, so some people are calling for the end of build steps.

When it comes to structuring CSS, I like to follow Andy Bell (https://piccalil.li/ and https://buildexcellentwebsit.es) who has some great ideas and insight when it comes to CSS.

Htmx

Brilliant little framework/tool for anyone who hates writing Javascript. Jokes aside, by placing a couple of extra attributes in your HTML you can entirely drop all boilerplate code that handles clicking on elements, getting some data, sending it to the server, getting something back and updating something in the DOM based on that data. So like 80% of your application's JS.

Been playing with it for a while and haven't found its limits yet, so very promising.

SQLite

It's fast, it's ACID compliant, the new wal2 mode is great. Yes, you can use it in production. The backups are dead simple, just rsync the files. Don't use it when you have to shard or support 10000 concurrent writers. Other than that, it's the GOAT.

Self written PHP framework

It's tiny, it's a single file, it has routing, templating, DI container and PSR-7 support. It's good enough for small apps: https://github.com/sanderdlm/mono

I use Doctrine DBAL to interact with my SQLite db, I avoid the ORM if I can. If I need auth I'll use the oauth2 bundle from thephpleague.

Cloudflare

Unbeatable free tier. Great domain registrar and DNS host. Free email forwarding. DDos protection too. Just works, and doesn't bother you.

DigitalOcean

AWS has managed to make their interface/system so complex, they need a ChatGPT wrapper to explain to you how to perform basic tasks. Absolutely vile product.

DO seems to tick the same boxes for me, without being repulsed when using it. It probably doesn't scale as well, but it's fine for my baby apps.
